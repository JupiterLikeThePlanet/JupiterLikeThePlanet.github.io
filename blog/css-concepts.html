What are the best practices associated with using classes vs. ids?

The simple breakdown is this:

  1. CLASSES should be used for common elements (like 'p','div','a') on a page that will likely be repeated
  2. IDs should be used for a unique page element. No two elements should share an ID

Let's look at an example of a good time to use a CLASS. Let's say we want to make the headers of each section to be bold and have the font, Garamond, and we want the paragraphs below to be in the font Helvetica.
