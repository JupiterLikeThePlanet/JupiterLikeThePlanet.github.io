Arrays

  • concat(): combine string or arrays
  • slice(): returns an array based on the range of indices
  • join() : combines elements in an array
  • sort() : sorts arrays in alphabetical order
  • length : lists the length in a string or index of an array
  • splice() : adds and removes items from an array, returning the removed ones
  • pop() : remove last element in an array
  • push() : add an element to the end of an array
  • toString(): convert content to string
  • reverse(): reverse elements of an array

Functions

A function can store variable/value or perform tasks. Parenthesis required, though parameters don't need to be set. Functions follow this format:

  • function name_of_function (arguments) {
    statements to execute when
    function is called;
    }

Iterations

There are couple of ways to iterate through an array. The first presented here is a sequential for iteration. It creates a counter using "i=0", saying as long as i is less than the length of the array (or string, etc.) run the code block and at the end of each block add 1 to i.

  • for (var i = 0; i < arrayLength; i++) { alert(myStringArray[i]);}

Then there's the for...in loop. In the example below, it's stating for variable key in array do the following code block.

  • for (var key in array) {}

Numbers

  • Math.floor() : rounds decimal down to nearest integer
  • Math.random() : returns a random value betweem 0 & 1

Comparison

  • == : returns true if the operands are equal
  • != : returns true if the operands are not equal
  • === : returns true if the operands are equal and the same data type
  • !== : returns true if the operands are not equal and/or not the same data type

Strings

  • indexOf: returns the position of the first occurrence of a specified value in a string
  • length : lists the length in a string or index of an array
  • split(): splits a string into an array
  • toLowerCase(): brings type to lower case
  • toUppperCase(): brings type to upper case
